## Module A: 表演调度深化

本模块扩展核心版 3.3 节，提供更精细的表演拆解方法论。

### A.1 面部表情——肌肉组拆解法

将面部分为五个控制区域，每个区域独立描述：

```
┌──────────────┐
│ ① 额头/眉毛  │ 上扬=惊讶/疑问  内收=愤怒/专注  平展=平静
│ ② 眼部       │ 睁大=震惊/恐惧  微眯=怀疑/得意  半闭=疲惫/沉醉
│ ③ 鼻部       │ 鼻翼张开=愤怒/喘息  皱鼻=厌恶  正常=中性
│ ④ 嘴部       │ 上扬=喜悦  下拉=悲伤  抿紧=隐忍  张开=惊讶/呐喊
│ ⑤ 下颌/颈部  │ 紧咬=压力  颤抖=恐惧/哭泣  前伸=挑衅  后缩=退避
└──────────────┘
```

**混合表情处理流程**：
1. 识别混合情绪的成分（如"强忍泪水的微笑"= 悲伤 + 勉强的喜悦）
2. 将每种成分分配到不同面部区域
3. 描述区域之间的矛盾张力

示例——"强忍泪水的微笑"：
```
嘴角向上拉出弧度但肌肉僵硬不自然，眼眶泛红蓄满泪水，睫毛微微颤动，眉头中部轻微上挑形成悲伤纹路，下颌肌肉紧绷控制住颤抖
```

### A.2 肢体姿态——三锚点法

每个姿态描述必须锚定三个参数：

**锚点①：重心**
- 重心在哪只脚 / 哪个支撑面
- 重心偏移方向暗示心理倾向（前倾=主动/攻击，后倾=被动/防御，侧倾=犹豫/闪避）

**锚点②：脊柱线**
- 挺直/前弯/后仰/侧弯/扭转
- 脊柱弧度直接表达角色当下的心理能量水平

**锚点③：手部状态**
- 手是"第二张脸"：手指的张合、力度、接触方式都传递信息
- 描述精度：指节级别
  - ❌ 握拳 → ✅ 五指缓慢蜷曲收紧，指甲陷入掌心，拇指压在食指第二关节上
  - ❌ 摊手 → ✅ 五指自然张开，掌心朝上，手指微微弯曲呈放松状态

### A.3 视线设计——控制观众注意力

**视线方向编码**：
```
        上
        ↑
  左上 ↗   ↖ 右上
左 ←    ●    → 右    （角色面向观众时的方向）
  左下 ↙   ↘ 右下
        ↓
        下
```
- 用时钟方位描述："视线指向两点钟方向"
- 或用画面参照："目光越过镜头左侧边缘，投向画面外"

**视线焦点状态**：
- 实焦：目光锁定具体对象，瞳孔聚焦，眼球稳定
- 虚焦/失焦：目光放空，瞳孔散大，不追踪任何物体
- 搜索：眼球快速小幅移动，在某个区域内寻找目标
- 闪避：视线短暂接触后快速移开

**视线的叙事功能**：
- 两个角色的视线交汇 → 对抗或连接
- 角色看向画面外 → 暗示画面外有信息，引导观众好奇心
- 角色直视镜头 → 打破第四面墙，制造压迫感或亲密感

### A.4 微动作库

| 微动作类型 | 情绪含义 | Prompt 描述 |
|-----------|---------|-----------|
| 吞咽 | 紧张/恐惧/压制情绪 | 喉结上下滚动一次 |
| 屏息 | 震惊/专注/屏气凝神 | 胸腔停止起伏，身体完全静止 |
| 叹气 | 疲惫/释然/无奈 | 肩膀先微微上提，随后缓缓塌落，气息从鼻腔呼出 |
| 频繁眨眼 | 紧张/不安/撒谎 | 眼皮快速连续闭合三到四次 |
| 停止眨眼 | 震惊/出神/恐惧僵直 | 眼睛保持睁开不眨，目光定住 |
| 嘴唇抿动 | 犹豫/思考/控制发言 | 双唇轻轻向内抿合再松开 |
| 手指轻敲 | 焦躁/等待/思考 | 食指和中指交替轻叩桌面 |
| 无意识摸脸 | 不安/自我安抚 | 手指无意识地沿脸颊滑过 |
| 攥紧衣角 | 紧张/害怕/不安 | 手指蜷曲抓紧衣服下摆，指节泛白 |
| 身体微晃 | 犹豫/不稳/眩晕 | 身体轴线轻微左右摇摆 |

## Module B: 光影色彩转译

### B.1 情绪→光影 映射表

| 情绪关键词 | 光影方案 | Prompt 描述 |
|-----------|---------|-----------|
| 压抑/绝望 | 顶光，高对比，深重阴影 | 正顶光照射，面部形成浓重的眼窝阴影，背景几乎全黑 |
| 温暖/安宁 | 黄金时段侧光，柔和阴影 | 暖金色侧光从窗口洒入，阴影边缘柔和过渡 |
| 紧张/不安 | 不稳定光源，冷暖交织 | 忽明忽暗的闪烁光源，冷蓝与暖橙在面部交替 |
| 神秘/未知 | 逆光/剪影，大面积暗部 | 人物逆光呈剪影状态，仅边缘轮廓有一圈光晕 |
| 愤怒/爆发 | 强硬侧光，红橙暖色主导 | 强烈侧光切割面部，一半亮一半暗，暖红色调笼罩 |
| 悲伤/孤独 | 散射冷光，低饱和 | 阴天散射冷光，整体色彩低饱和偏灰蓝 |
| 希望/觉醒 | 暗中光束，暖调渐入 | 一束暖光从黑暗中穿透而来，照亮人物面部 |
| 恐惧/惊悚 | 底光，不完整照明 | 从下方打出的光源，面部阴影向上扭曲，背景漆黑 |
| 甜蜜/浪漫 | 柔焦暖光，微光粒子 | 柔和暖光笼罩，空气中飘浮着细微光点 |
| 决绝/壮烈 | 硬朗逆光，高光溢出 | 强烈逆光在人物周围形成燃烧般的光晕 |
| 日常/平和 | 自然光，中性色温 | 自然日光均匀照射，阴影柔和，色彩还原真实 |
| 癫狂/失控 | 多方向混乱光源，高饱和 | 来自多个方向的彩色光源交叉照射，色彩浓烈刺眼 |

### B.2 色彩心理参考

| 色调 | 心理暗示 | 适用场景 |
|------|---------|---------|
| 暖金/琥珀 | 怀旧、温情、回忆 | 回忆段落、亲情场景 |
| 冷蓝/青灰 | 疏离、理性、孤独 | 都市夜景、孤独角色 |
| 红/深红 | 危险、激情、暴力 | 冲突高潮、情欲场景 |
| 绿/墨绿 | 自然、诡异、腐朽 | 森林、实验室、病态场景 |
| 紫/靛 | 神秘、高贵、超自然 | 魔法场景、权力象征 |
| 橙/暖橙 | 活力、焦虑、紧迫 | 夕阳、爆炸前、追逐 |
| 低饱和/消色 | 绝望、虚无、死亡 | 战争场景、角色死亡 |
| 高饱和 | 梦境、幻觉、癫狂 | 超现实段落、毒品幻觉 |

### B.3 动态光影设计

部分镜头的光影在镜头进行过程中需要发生变化。在 Prompt 中描述光影变化的格式：

```
起始光影状态 → 触发点 → 终止光影状态
```

示例：
```
画面起始时房间昏暗仅有冷蓝色月光，角色推开窗帘，暖金色晨光涌入照亮面部
```

## Module C: 景别与运镜 Prompt 词典

### C.1 景别词典

| 景别术语 | 可灵 Prompt | 即梦 Prompt |
|---------|-----------|-----------|
| 大远景 | 超远景，极广角，人物如蚁点般渺小 | 超远景全景，广袤空间，人物极小 |
| 远景 | 远景，展示完整环境和人物全貌 | 远景镜头，环境全貌，人物可辨 |
| 全景 | 全景，人物从头到脚完整可见 | 全身镜头，人物完整站立于画面中 |
| 中景 | 中景，人物膝盖以上 | 中景，腰部以上，交代动作和表情 |
| 中近景 | 中近景，人物胸部以上 | 中近景，胸部以上，强调表情和对话 |
| 近景 | 近景，人物肩部以上 | 近景镜头，肩部以上，聚焦面部表演 |
| 特写 | 特写，面部充满画面 | 面部特写，五官细节清晰可见 |
| 大特写 | 大特写，局部细节填满画面 | 超特写，仅展示眼睛/嘴唇/手部等局部 |
| 过肩镜头 | 过肩镜头，前景为背面肩膀，焦点在对面人物 | 过肩视角，一方肩膀做前景虚化框 |
| 主观镜头 | 第一人称视角，镜头即角色的眼睛 | 主观视角镜头，画面即角色所见 |

### C.2 运镜词典

| 运镜术语 | 可灵 Prompt | 即梦 Prompt |
|---------|-----------|-----------|
| 缓推 | 镜头缓慢向前推进 | 缓慢推近，镜头逐渐靠近主体 |
| 快推 | 镜头快速推进至主体 | 快速推进，镜头迅速逼近 |
| 缓拉 | 镜头缓慢向后拉远 | 缓慢拉远，镜头逐渐远离主体 |
| 急拉 | 镜头猛然快速拉远 | 急速拉远，瞬间揭示宏大环境 |
| 左横摇 | 镜头水平向左摇动 | 水平向左摇镜 |
| 右横摇 | 镜头水平向右摇动 | 水平向右摇镜 |
| 上摇 | 镜头垂直向上摇动 | 镜头从下往上缓缓摇起 |
| 下摇 | 镜头垂直向下摇动 | 镜头从上往下缓缓摇落 |
| 环绕 | 镜头围绕主体 360 度旋转 | 环绕镜头，摄影机围绕主体旋转 |
| 跟随 | 镜头跟随角色运动保持距离 | 跟随镜头，与角色同步移动 |
| 手持 | 手持镜头，画面带有轻微晃动 | 手持摄影风格，微微抖动的镜头 |
| 固定 | 固定镜头，完全静止不动 | 固定机位，镜头纹丝不动 |
| 升起 | 镜头垂直向上升起 | 摇臂升起，从低位升至高位俯瞰 |
| 降下 | 镜头垂直向下降落 | 摇臂降落，从高位下降到人物平视 |
| 甩镜 | 镜头快速甩动到另一目标 | 快速甩镜，画面在两个焦点间急速切换 |
| 变焦推 | 不移动机位，变焦拉近主体 | 变焦推近，压缩空间透视感 |
| 斯坦尼康 | 镜头平稳流畅地跟随或穿越空间 | 稳定器长镜头，流畅穿越空间 |

### C.3 特殊视角词典

| 视角术语 | Prompt 描述 |
|---------|-----------|
| 仰拍 | 低角度仰拍，镜头从地面向上拍摄，人物高大 |
| 俯拍 | 高角度俯拍，镜头从上方向下拍摄，人物渺小 |
| 鸟瞰/正俯 | 正上方垂直向下俯瞰，呈地图式构图 |
| 蚁视 | 极低角度贴地仰拍，如蚂蚁的视角 |
| 荷兰角 | 镜头倾斜 15-30 度，画面歪斜，不安感 |
| 窥视 | 前景有遮挡物（门缝/树叶/铁栏），如偷窥视角 |
