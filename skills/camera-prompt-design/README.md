# AI 导演工作流 · 第二层：镜头画面设计与 Prompt 工程

## 这是什么

这是一套让 AI 扮演"视觉总监"的 Prompt 系统。它接收第一层（导演分镜脚本）产出的分镜表格，把每个镜头从"导演想要什么"翻译成"AI 视频工具能执行什么"——包括首帧参考图 Prompt、可灵/即梦双平台视频 Prompt、表演调度指令和镜头分组建议。

简单说：**分镜表进去，可直接喂入生产工具的完整指令包出来。**

---

## 文件说明

| 文件 | 作用 | 是否必须 |
|------|------|---------|
| `director-visual-design-core.md` | 核心版。覆盖从分镜表到可执行 Prompt 的完整主干流程 | ✅ 必须 |
| `director-visual-design-advanced.md` | 进阶版。提供表演词库、光影映射、运镜词典、多工具适配等扩展模块 | 可选 |

**只加载核心版就能正常工作。** 进阶版是弹药库，按需加载，让输出更精细。

---

## 使用前提

1. 你已完成第一层，手上有一份完整的分镜表格（8 列格式）
2. 你知道自己打算用什么工具生成视频（可灵 / 即梦 / 都试试）
3. 你大致知道想要的美术风格（写实？动画？赛博朋克？——不确定也没关系，系统会引导你选）

---

## 怎么用

### 第一步：加载

将 `director-visual-design-core.md` 粘贴到你的 AI 对话中（作为系统提示或直接发送均可）。如果需要进阶功能，同时加载 `director-visual-design-advanced.md`。

### 第二步：喂入分镜表

把第一层产出的分镜表格发给 AI。可以一次性全部发送，也可以分场次发送。

### 第三步：确认视觉基准（Phase 0）

AI 会先问你几个全局设定问题：
- 美术风格（写实/动画/其他）
- 画面比例（16:9 / 9:16 / 2.35:1）
- 主色调、光影基调
- 首帧参考图用什么工具生成（可灵文生图 / 即梦文生图 / Midjourney / SD / 其他）

不确定的项，AI 会给推荐值，你确认就行。

### 第四步：确认角色和场景资产卡（Phase 1-2）

AI 会为每个主要角色和场景生成视觉资产卡，包含：
- 详细的外貌/环境描述
- 角色参考图 Prompt（用于生成一致性参考图）
- 平台引用标签占位（生成参考图后回填）

**检查重点：** 角色长相和服装描述是否符合你的想象。这一步定下来后，所有镜头都会引用这些卡片，改起来成本很高。

### 第五步：逐镜头生成（Phase 3）

AI 会按每批 10-15 个镜头的节奏，输出每个镜头的完整指令包：
- 🖼 首帧参考图 Prompt
- 🎬 可灵视频 Prompt
- 🎬 即梦视频 Prompt
- 🎭 表演调度（面部/肢体/视线/微动作）
- 🔊 音频标注
- ⚠️ 风险提示（如有）

每批输出后 AI 会暂停，等你确认后继续下一批。

### 第六步：镜头分组建议（Phase 4）

全部镜头处理完后，AI 输出分组建议表——告诉你哪些镜头可以合并生成、哪些必须独立生成、每组推荐用哪个平台。

## 进阶版能额外做什么

不加载进阶版时，核心版已经能产出完整可用的 Prompt。

加载进阶版后，AI 在生成过程中会自动调用以下模块来提升输出质量：

| 模块 | 能力 |
|------|------|
| A 表演深化 | 面部肌肉组拆解、三锚点肢体法、视线编码系统 |
| B 光影色彩 | 情绪→光影自动映射、色彩心理参考、动态光影设计 |
| C 运镜词典 | 景别/运镜/视角的双平台对照 Prompt 词典 |
| D 工具适配 | Midjourney、Stable Diffusion、Flux 等首帧工具的 Prompt 自动转化 |
| E 平台优化 | 可灵 Elements 策略、即梦四模态组合、平台选择决策树 |
| F 视觉词库 | 15 种表情、12 种姿态、10 种光影预设的即用 Prompt |
| G 高级分组 | 正反打序列、蒙太奇、长镜头、闪回段落的专项分组规则 |
| H 边界处理 | 手部风险分级、多角色同框、特效镜头、文字叠加的应对策略 |
| I 质量检查 | 20 项自检清单，覆盖 Prompt 质量/视觉一致性/表演/平台适配/风险 |

## 常见问题

**Q：我只用可灵，不用即梦，还会输出两个平台的 Prompt 吗？**

A：会。双平台 Prompt 都会输出，你只用你需要的那个就行。如果你明确告知只用某个平台，AI 会只输出对应格式。

**Q：首帧参考图我想用 Midjourney 生成，Prompt 会自动转英文吗？**

A：加载进阶版后会自动转化。只用核心版时，输出的是通用中文格式，你需要自己翻译或告诉 AI 帮你转。

**Q：分镜表格式不是标准 8 列怎么办？**

A：把你的分镜表直接发过来，AI 会尝试识别并适配。但标准 8 列格式效果最好。

**Q：中途想改一个角色的造型怎么办？**

A：告诉 AI 修改角色资产卡，然后要求从受影响的第一个镜头开始重新生成。已确认的镜头不会自动更新。

**Q：我能只处理某几个镜头吗？**

A：可以。告诉 AI 你要处理的镜头编号范围即可。
