## Seedance 2.0 适配

### 特点
- 原生 2K 分辨率
- 支持多镜头叙事（>1 分钟）
- 原生音视频同步
- 导演级控制（运镜、光线、构图）

### Prompt 格式
```
[场景描述] + [运镜指令] + [光线/色彩] + [情感基调]

示例：
清晨的城市天际线，相机缓慢横移展现玻璃幕墙建筑群，
柔和的晨光侧逆光照射，冷色调带有科技感和未来感，
画面简洁大气，商务专业氛围
```

### 多镜头提示词模板
```
镜头 1（0-5 秒）：[画面描述]，运镜：[运镜方式]
镜头 2（5-10 秒）：[画面描述]，运镜：[运镜方式]，保持：[需要一致的元素]
镜头 3（10-15 秒）：[画面描述]，运镜：[运镜方式]，保持：[需要一致的元素]
```

### M2V 记忆注入格式
```
记忆注入：
- 角色：[服装、发型、配饰描述]
- 场景：[背景、道具、光线描述]
- 产品：[颜色、材质、Logo 位置描述]
- 上一镜头关键帧：[关键视觉元素]
```

## 多镜头连贯性实现

### M2V（Memory-to-Video）机制

这是实现多镜头角色和场景连贯性的核心技术。

#### 工作原理

1. **关键帧提取**
   - 每个镜头生成后，自动提取关键视觉元素
   - 包括：主体特征、场景元素、光线方向、色彩调性

2. **记忆库构建**
   ```
   记忆库 = {
       "角色": {
           "服装": "白色衬衫，深灰色西装外套",
           "发型": "短发，黑色",
           "配饰": "银色智能手表"
       },
       "场景": {
           "背景": "现代办公室，玻璃幕墙",
           "道具": "白色办公桌，黑色椅子",
           "光线": "自然光，左侧 45 度角"
       },
       "产品": {
           "颜色": "银色表壳，黑色表盘",
           "显示": "蓝色 UI 界面"
       }
   }
   ```

3. **记忆注入**
   - 生成下一镜头前，将记忆库内容注入 prompt
   - 使用"保持"/"延续"/"一致"等关键词强化连贯性

4. **一致性检查**
   - 生成后比对关键元素是否一致
   - 如不一致，调整 prompt 重新生成

### 实现策略

#### 策略 1：详细角色设定
```
角色设定（全片保持一致）：
- 姓名：张明
- 年龄：30 岁左右
- 服装：浅蓝色衬衫，袖口卷起，深灰色西装裤
- 发型：短发，侧分，黑色
- 配饰：银色智能手表（左手腕）
```

#### 策略 2：场景锚点
```
场景锚点（贯穿全片）：
- 地点：现代化办公室
- 背景元素：玻璃隔断墙，绿植，白色办公桌
- 光线：上午自然光，从左侧落地窗射入
- 色彩基调：冷色调，灰白蓝为主
```

#### 策略 3：产品一致性
```
产品规范（每个镜头必须一致）：
- 型号：SmartWatch Pro X1
- 颜色：午夜黑表壳，黑色氟橡胶表带
- 屏幕：圆形 AMOLED，直径 46mm
- Logo 位置：表盘 12 点位置，表冠 3 点位置
- UI 主题：蓝色数字表盘
```

## Prompt 优化技巧

### 1. 具体化描述
❌ 不好：一个人在办公室
✅ 好：30 岁商务男士，浅蓝色衬衫，坐在现代办公室的白色办公桌前，左手腕佩戴银色智能手表

### 2. 光线指定
❌ 不好：光线很好
✅ 好：柔和的自然光从左侧 45 度角射入，在桌面形成柔和阴影

### 3. 运镜指令
❌ 不好：相机移动
✅ 好：相机从中景缓慢推近至手表特写，速度均匀

### 4. 情感基调
❌ 不好：看起来很专业
✅ 好：商务专业氛围，冷色调，简洁高效，高端质感

### 5. 连贯性强化
在后续镜头 prompt 中明确说明：
```
保持以下元素与前一镜头一致：
- 角色服装：浅蓝色衬衫，深灰色西装裤
- 手表位置：左手腕，银色表壳
- 场景光线：左侧自然光，45 度角
```
